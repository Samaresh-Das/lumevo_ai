# Lumevo AI

Your AI productivity suite — write smarter, grow faster.

## Products

- **CovGen** (Live) — AI cover letter writer. Paste a job description → AI generates a tailored, ATS-optimized, human-sounding cover letter in seconds.
- **DailyContent** (Coming Soon) — Agentic AI that creates and uploads content on your behalf to LinkedIn, Instagram, and more — fully autonomous.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS v4
- **Language**: JavaScript + TypeScript (components)
- **Animations**: Framer Motion, GSAP-lite via CSS
- **3D**: Spline (hero background)
- **Particles**: tsparticles

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
